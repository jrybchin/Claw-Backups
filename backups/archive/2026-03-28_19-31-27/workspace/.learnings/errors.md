# Error Log

This file tracks recurring errors, failures, and agent mistakes. 
When an error occurs, log it here. If an error reaches 3+ recurrences, it MUST be escalated for a systemic fix.

## Format
```markdown
### [ERR-YYYYMMDD-01] Short Description
- **Date:** YYYY-MM-DD
- **Priority:** High | Medium | Low
- **Status:** Open | Resolved | Promoted
- **Recurrence:** 1
- **Area:** (e.g., Memory, Integration, Logic)
- **Description:** What happened? Why did it fail?
- **Action/Fix:** What is the proposed or applied fix?
```
